Bienvenue !

Run Report est un site qui vous permet de gérer vos entrainements de course à pied, mais surtout de les partager en direct avec votre entraîneur !

Se connecter Créer un compte

Pour les athlètes

Vous pouvez gratuitement constituer votre carnet d'entrainements, planifier vos séances, voir concrètement votre évolution sportive et communiquer avec votre entraîneur.

Pour les clubs

Ce site vous fournit un moyen simple et complet pour gérer vos athlètes, répartis sur plusieurs entraineurs.
Chaque entraineur peut consulter les séances de ses athlètes, créer des plans d'entrainements, et communiquer efficacement avec eux.

Synchronisation Garmin©

Si vous disposez d'une montre Garmin, vos données peuvent être automatiquement récupérées (distance, temps, ...) et intégrées à vos statistiques de séances.
